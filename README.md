# GameStore Web Service API

UAS Mata Kuliah **Interoperability**  
Tema: **Online Game Store (Steam-like Platform)**

## Deskripsi
GameStore Web Service API adalah layanan backend berbasis REST API
yang dikembangkan menggunakan PHP Lumen. Sistem ini menyediakan
layanan manajemen game, user, transaksi, dan review untuk platform
penjualan game online.

## Teknologi yang Digunakan
- PHP Lumen
- MySQL
- JWT Authentication
- RESTful API
- Postman
- GitHub
